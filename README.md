# Face2R

Real-time face recognition from your phone.

**Live:** [https://eejimkos1.github.io/Face2R/](https://eejimkos1.github.io/Face2R/)

## Features

- Real-time face detection and recognition via phone camera
- Add new faces instantly by tapping
- Auto-syncs across all connected devices in real-time
- Risk classification (Low / Neutral / Risk)
- Confidence scoring with photo diversity analysis
- PIN-protected access
- PWA — install as an app on your phone
- Works offline after first load
- Optimized for indoor/evening conditions
